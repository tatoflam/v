## 13:36  wiki 自動運用パイプライン構築（hook + CI）
- session: 8a25326c  cwd: v
- 目標：「セッション終了 → 自動 ingest → vault に push」を全自動化
- 採用した最終構成：
  - **plugin の SessionEnd hook**：従来通り queue.jsonl に enqueue するだけ
  - **ユーザ設定の SessionEnd hook**（追加）：`~/.claude/wiki/hooks/auto-ingest-push.sh` → `claude --dangerously-skip-permissions -p "/wiki-ingest"` → `git push`、`WIKI_INGEST_WORKER=1` で再帰防止

- VS Code 拡張でも `SessionEnd` はパネル閉じ・別セッション切替で発火する（`/exit` 必須ではない）ことを確認
- 設定変更後の hook はそのセッションには効かない（セッション開始時にロードされる）
