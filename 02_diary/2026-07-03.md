### 工程チャート renderer デグレ（今後の再発防止メモ）

`plugins/drive-publisher/scripts/render_html.py` の `parse_phase_progress` は各工程行が `- {N}. {工程名}: {内容}` の**ダッシュ箇条書き**であることを `_PHASE_HEADER_RE = ^-\s+\d+\.…` で要求する。ダッシュを外した番号リストにするとパターン不一致で HTML 側は「工程ブロックチャート」ではなく **生 Markdown フォールバック**に落ちる（15 phase-row → 0 phase-row）。今回はセッション内で発見 → 修正 → 再 publish で復旧、[[memory:project_drive_publisher_renderer_coupling]] の「HTML レンダラは見出しレベル・箇条書き形式に依存」に該当。**markdown を弄って renderer と食い違わせない、または renderer を先に見て markdown 側の書式を合わせる** のを維持。
