# エミル株式会社 法人運営 (節税・投資スキーム)

## Summary

事業法人エミル株式会社 (法人 3 期目、本業利益あり、税務顧問 Take Off 佐藤氏) の節税・投資スキーム検討ページ。資産管理法人ソルトムーンとは役割を分け、**節税スキーム (損金算入を伴う投資案件) はエミル側で運用** することを基本方針とする。詳細経緯と背景は [[04_life/saltmoon-llc-operations]] 参照。

## 現在の状態

700 万円帯オペレーティングリース・節税案件のリサーチ完了 (2026-06-12)。運用主体はエミルで確定済、業者への資料請求・試算依頼は未着手 (下記「次の一手」)。

### 次の一手 (実行計画)

1. **トレーラーハウス 1〜2 業者** に「エミルでの法人取得・節税効果シミュレーション付き」で資料請求
   - R&I 方式 900 万円案件
   - 500 万円台 小型案件
2. **キャンピングカー運用代行 2 業者** に同様の試算依頼
   - 車インベスト
   - 税務キーパー
3. **見積もりが揃ったら Take Off 佐藤氏にぶつけて** 否認リスク + 実効税率での節税額 + キャッシュフロー試算を依頼
4. エミル側の **今期着地見込み (本業利益)** を確認して、損金算入したい額を確定 (= 投資規模の上限設定)

### 留意点 (オペレーティングリースの一般的リスク)

- **否認リスク**: 国税庁の「経済合理性を欠く」判定で減価償却 / 即時償却が否認される可能性 (= 過去の航空機リース判例多数)
- **流動性低**: 中古売却市場が薄く、想定利回りを下回るケースあり
- **運用代行手数料率**: 表面利回りから 10-15% 控除した実質利回りで判断
- **法人 3 期目特有**: 黒字化初期で節税余地がまだ小さい可能性 → 「投資のために借入をして節税」が逆効果になるパターンに注意

## 決定事項

- 2026-06-12: 節税スキームの運用主体は **「やるならエミル」** (user 決断)、ソルトムーンは資産管理 (= 不動産取得) に専念 — エミルは本業利益と損金を相殺可能、ソルトムーンには損金を吸わせる利益が不在のため。
- 2026-06-12: 700 万円帯は **JOL/JOLCO 不適**と判断、車両系・小規模設備系の即時 / 短期償却スキームに絞る — 伝統的な日本型オペレーティングリース (JOL/JOLCO) の最低出資額はコンテナ系 1,000 万円〜、航空機・船舶系 3,000〜5,000 万円〜で 700 万円帯は最低出資未満のため。
