#### 第4期以降の見通し

- 売上は年¥5.5-6.0M でほぼ横ばい（賃料下落リスク年▲1-2%）
- 減価償却は **附属設備分割しないと年¥3M固定**、22年で建物本体終了
- 借入利息は元金返済進行で漸減
- 黒字化転換は**第5-6期前後**と想定（繰越欠損金10年消化中なら法人税は均等割のみ）
- **建物附属設備分割（電気¥7M／給排水¥5M／空調¥3M等を15-6年償却）** で初期数年の減価償却を加速 → 繰越欠損金の積み増し効果を最大化（佐藤氏との初期論点）

#### シミュレーションの感応度

**家賃水準による収益変動**:
| シナリオ | 平均賃料 | 稼働率 | 第3期売上 | 営業損益 |
|---|---:|---:|---:|---:|
| 強気（Riel想定） | ¥60k | 95% | ¥6.16M | +¥0.4M |
| 中立（本シミュ） | ¥55k | 90% | ¥5.40M | ▲¥1.3M |
| 弱気（ヤマト地所想定） | ¥45k | 85% | ¥4.13M | ▲¥2.7M |

**最重要監視指標**: 第3期下期（2028年春）の **再募集時賃料** と **稼働率**。ここが弱気シナリオに収束すると累計欠損金が膨らみ、AP買い増し時の融資審査に影響する可能性あり。

#### シミュレーション結論
- 第1期〜第3期は **構造的赤字** を前提に運営
- 法人税は **均等割¥70k／年のみ** で着地
- **繰越欠損金10年** を将来の黒字化フェーズで活用（節税策の温存）
- **法人税の心配は実質ない** ため、無理な節税策（経営セーフティ共済等）は不要

### E. 創立費・開業費の繰延資産処理

#### 定義

**創立費（会社法上の創立費）**
- 会社設立のために発生した費用
- 例：定款作成費（電子定款¥0 or 紙¥40k）、定款認証費、設立登記の登録免許税（合同会社¥60k）、印鑑作成費、設立時の交通費、司法書士報酬、設立時の広告費

**開業費**
- 法人設立後、営業開始までに発生した特別な費用
- 例：開業前の事務所家賃・通信費、開業案内費、市場調査費、開業前の打合せ交通費・接待交際費
- ※ 経常的な費用（毎月の電気代等）は開業費に**含めない**

#### 処理方法

| 項目 | 会計上の原則 | 税法上の特例 |
|---|---|---|
| 償却期間 | 5年以内の均等償却 | **任意償却（5年以内のいつでも全額OK）** |
| 仕訳 | 繰延資産として資産計上 | 同左 |
| 償却時 | 創立費償却・開業費償却（営業外費用） | 損金算入 |

**税法上の任意償却が最大の節税武器**:
- 売上が出るまでは資産計上のまま温存
- 黒字化のタイミングで全額一括費用化 → 黒字相殺

#### ソルトムーンでの想定処理

**第1期に発生した創立費・開業費**（推定）:

| 項目 | 金額 | 区分 |
|---|---:|---|
| 設立登記の登録免許税 | ¥60,000 | 創立費 |
| 定款認証費（電子定款の場合¥0） | ¥0 | 創立費 |
| 印鑑作成費（実印・銀行印・角印3点） | ¥30,000 | 創立費 |
| 司法書士報酬 | ¥80,000 | 創立費 |
| 印鑑証明書・登記事項証明書取得 | ¥10,000 | 創立費 |
| 設立時の交通費（法務局・公証人役場） | ¥5,000 | 創立費 |
| 物件選定の調査費・交通費（設立後分） | ¥30,000 | 開業費 |
| 銀行口座開設の交通費 | ¥5,000 | 開業費 |
| 名刺・社判の作成 | ¥20,000 | 開業費 |
| **合計** | **約¥240,000** | |

#### 仕訳例

**第1期 発生時**:
```
（借）創立費  170,000  （貸）現金預金  170,000
（借）開業費   55,000  （貸）現金預金   55,000
```
→ 損益計算書には影響なし、貸借対照表の「繰延資産」に資産計上

**第3-5期 黒字化時に一括償却**:
```
（借）創立費償却  170,000  （貸）創立費   170,000
（借）開業費償却   55,000  （貸）開業費    55,000
```
→ 営業外費用として計上、当期利益を圧縮

#### 注意点

1. **領収書・請求書をすべて保管**（税務調査対応、最低7年）
2. **設立日の前後で区分**：
   - 設立日**前**の費用 → 創立費
   - 設立日**後・営業開始前**の費用 → 開業費
   - 営業開始**後**の費用 → 通常の経費（広告宣伝費・旅費交通費等）
3. **不動産投資の場合の「営業開始」とは**：
   - 一般的には **最初の家賃発生日**（賃貸借契約締結 or 入居開始）
   - 引渡前の物件取得関連費用 → 開業費の余地あり（税理士判断）
4. **入居者募集の広告費**（AD・仲介手数料）は **開業費ではなく** 各期の「広告宣伝費」「支払手数料」
5. **不動産取得税・登録免許税（建物・土地）** は創立費ではなく **取得価額に算入 or 一括費用化** の選択（取得価額算入の方が将来の譲渡時に有利な場合あり）
