# 狭山市土地売買・建築請負契約

## Summary
埼玉県狭山市富士見一丁目 15-18（地番 6044-9）で、土地先行方式により土地売買契約＋建築請負契約を **2026-04-28(火) 10:00 新橋 Riel社** にて締結予定。物件 ¥89.6M・総事業費 ¥94.7M。売主 安藤建設、仲介コンサル Riel社（関根氏）。2026-04-23 に現地視察を実施し、Riel関根氏へ確認事項メールを2通送付済（回答期限 4/25 金）。

## 物件概要（S-068）
- 所在地：狭山市富士見一丁目 15-18（地番 6044-9）
- 敷地面積：106.49㎡（32.21坪）
- 用途地域：**第二種住居地域**、建蔽率 60%＋10%（角地）= 70%、容積率 法定200% / 道路幅員 4.5m×0.4 = **180% 適用**
- 接道：北東 4.5m・北西 4m の2面接道（角地）
- 建物計画：**木造3階 9戸（18㎡×6戸＋20㎡×3戸）**、延床 168.07㎡、施工床 215.32㎡
- 総事業費：**¥94.7M**（土地 ¥18.8M / 建物 ¥70.8M / 地盤・水道バッファ ¥4.0M / 諸経費 ¥1.1M 相当）
- 借入：¥78M / 35年 / 金利 2.55%、自己資金 ¥17.87M
- 入居開始予定：2027/6/5（着工8ヶ月想定）

## 現地視察所感（2026-04-23）

### 街の素性
- 狭山市駅は想定以上に商業充実：いなげや系列スーパー、タリーズ、ドトール、ミスド、亀屋、ちよだ寿司、権兵衛、ファミマ、セブン、マツキヨ
- 駅→現地 **徒歩約10分**、軽く上るが以降平坦
- 現地周辺：東にすき家（深夜営業・排気騒音リスク）、北に月極駐車場（オーナー自主管理・将来建替リスク）

### 敷地・境界
- 形状：北に約21歩・西に約10歩で認識（約 15m×7m 相当）
- **隣地佐藤邸の当地側窓は全閉鎖**（民法235条関連、新築時に目隠し設置要求の可能性あり）
- ガス・水道管は敷地内に既設引込あり（前戸建て時代のものと推測）
- 2段ブロック盛土で GL 確保、隣地同水準

## 収益性の再評価

### 実勢利回りの再試算
- Riel想定：表面利回り **7.26%**（満室・初期賃料ベース、空室・賃料下落・修繕費・AD 未織込）
- 実勢ベース：
  - 新築プレミアム＋オートロック込みで初期 56-59k は到達可能
  - 2-3年後の再募集時は **50-53k 台**に下落、稼働率 **85-90%** が現実的
  - 安定稼働賃料 約 510k/月・6.12M/年
  - **実勢表面利回り 6.83%（満室）／ 6.38%（諸費用込み）**
  - 運営費25%を引いて **ネット利回り 4.7-4.9%**
- **30年累積CF**: Riel試算 ¥30M → 実勢 **¥10-15M**（原状回復・AD・修繕積立・空室率を加算補正）

### 建築費の評価
- Riel提示 ¥70.8M / 50.84坪 = **¥1.393M/坪** → 木造3F 共同住宅の市場相場 ¥1.0-1.2M/坪と比べ **上限寄り**
- 相見積もりでの削減余地 **¥5-10M**
- ただし Riel は伴走（土地売買＋建築請負一体）モデルで**建築会社を切替えられない**前提 → 仕様の作り込みで勝負

### 土地価格の評価
- ¥18.8M / 32.21坪 = **¥583,915/坪（≒¥176,540/㎡）**
- 駅徒歩9分・角地・法規MAX使える形状 → 単品として妥当〜やや高め
- 要確認：2026 公示地価・路線価での裏取り
- 売主安藤建設の仕入推定 ¥13-15M（2026-01-30 取得、抵当権 ¥9M 埼玉縣信用金庫）→ 開発者マージン ¥4-6M（常識的水準）

## プラン改善の検討ログ（見送り判断を含む）

### プラン α：全戸ロフト付1K（**撤回**）
- 案：18㎡ + ロフト 6-9㎡ で実用 25-27㎡ 相当、賃料 +3-6k
- 撤回理由：
  1. **日影規制（4h-2.5h/4m）に抵触リスク大**（ロフト分で高さ+1.4-2m、冬至の影が4-8m 伸長）
  2. 階高を下げると3F天井 2.1-2.3m で圧迫感、商品性低下
  3. 18㎡ターゲット層（工業団地・病院単身者）に**ロフト不人気**（荷物の上げ下ろし忌避）
  4. 狭山市建築指導要綱の小屋裏物置規定の確認必要（条例で居室利用不可なら寝室ロフト不可）

### プラン β：6戸 1LDK/メゾネット化（**見送り**）
- 案：1F 1LDK 30㎡×2戸（庭付）+ 2-3F メゾネット 1LDK 35-40㎡×4戸
- 長期稼働率 94-96% 期待、退去頻度低下、EXIT 価格 +5-10%
- **見送り理由**：総賃料が下がる（利回り低下）、Riel が嫌がる前提、そもそも契約組成上 Riel プランからの大幅変更は困難

### プラン γ：ピロティ駐車＋3Fメゾネット（**見送り**）
- 案：1F ピロティ駐車 2台 + 2-3F メゾネット 1LDK 40㎡×3戸
- 駐車収入 +16k/月、差別化価値
- **見送り理由**：構造計算を一から実施し直すリスク、Rielが応じない前提、そもそも非現実的

### 結論
**Rielパッケージ前提で、仕様書・契約条項で攻める**方針に収束。
運営面では「家具家電付き・企業契約優先募集（工業団地・病院）・マンスリー対応」で 18㎡ の弱点を運営力でカバー。

### 1通目：契約前確認事項（A 資料開示 / B 契約条項 / C 売主確認）
**A. 資料開示（8項目）**
1. 建築費 ¥70,800,000 の項目別内訳
2. 地盤改良・水道引込バッファ ¥4,000,000 の内訳
3. ▲¥2,000,000 減額の前提条件
4. **抵当権設定登記費用の分割理由**（土地 ¥400k・残代金 ¥300k の分割根拠）
5. 日影図・天空率検討書・道路斜線検討書
6. 給水計画書（既設引込活用可否・口径・工事内容）
7. 建物仕様書（サッシ・断熱・遮音等級・設備メーカー型番・外壁/屋根材）
8. 近隣地盤データまたは過去の地盤調査事例

**B. 契約書ドラフトへの追加条項**
1. 地盤改良費・水道工事費の**実費精算**、上限 ¥4M、差額は買主還元
2. 改良費超過時（¥1M 以上）の**買主の契約解除権**
3. 既設水道管活用時の**減額規定**
4. 土地売買と建築請負の**連動特約**（建築確認不可・大幅設計変更時の解除権）
5. 境界・隣地関係の**告知書**（佐藤邸側窓経緯、すき家・駐車場オーナー既往）
6. 竣工時期と**遅延損害金**条項（着工後8ヶ月想定、入居開始予定 2027/6/5）
7. **瑕疵担保・アフター保証**（構造10年・仕上げ2年 以上）

**C. 売主（安藤建設）経由で確認事項**
1. 既設水道引込管の**口径・材質・設置年**、再使用可否
2. 隣地（佐藤邸）との**過去経緯**・境界同意状況
3. **擁壁ブロック2段**の扱い
4. 前所有者（三輪田氏）時代の**地中埋設物**告知
5. **接道道路隅切りの既存ゴミ置場**（管理主体／本物件入居者使用可否／町内会費／専用ストッカー要否／市ゴミ収集ルール上の位置付け）

### 2通目：建物仕様（C項単独）
1. 居室天井高 **2.5m以上**（日影規制の範囲で最大化）
2. 開口部を十分に確保（採光・通風・居室の広さ感に配慮）
3. オートロック／モニター付インターホン／宅配ボックス／無料Wi-Fi（仕様書明記）
4. 独立ミニ洗面／**2口ガスコンロ**（プロパン想定のためIH外）／室内洗濯機置場
5. 外壁：窯業系サイディング **16mm以上** 同等品
6. 共用廊下・階段：**FRP防水** グレード指定
7. **断熱等性能等級 4 以上**
8. **耐震等級**（等級2以上希望、地震保険料割引＋売却時有利）
9. **劣化対策等級**（等級3希望 ≒ 75-90年耐用）
10. **維持管理対策等級**（等級3で長期メンテコスト削減）
11. 建築確認申請前の**段階的なレビュー期間確保（複数回）**

## 要注視・要確認（契約時〜契約後）

- **業務委託手数料 ¥2,004,000**（仲介手数料と別建て）：土地価格¥18Mの約11%に相当、Riel の実質開発利益の一部の可能性。契約条項上は削除対象にせずとも、交渉の材料として意識
- **地盤改良期待値 ¥150-200M**（武蔵野台地 + 2段ブロック盛土で悪化バイアス）：契約書上 ¥4M バッファの清算方式が買主還元型かを最優先確認
- **既設水道管の再利用可否**：50-60年物・20mm 径の可能性大、鉛/鋼管なら法的全量更新必須 → 期待値 ¥50-100M の追加コスト
- **境界確定**：お客様資料 p.11 で済と記載（現況実測図完成）、契約時に実地確認
- **ローン特約**：融資不承認時の白紙解除条項の期限と条件
- **契約不適合責任**：範囲・期間
- **違約金**：通常 売買代金の20%
