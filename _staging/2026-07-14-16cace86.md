# ソルトムーン 厚生年金・健康保険 加入状況調査への申立書作成（2026-07-14）

年金機構から資産管理法人ソルトムーンに厚生年金保険・健康保険の加入状況調査の連絡。
未加入理由の回答文書を作成した。

## 論理構成（決定事項）

- 法人は原則強制適用なので「加入しない」ではなく **「被保険者となるべき者が存在しない」** の論理で回答する
  （報酬月額 0 円 → 標準報酬月額が算定できない → 健保法3条・厚年法9条の被保険者になり得ない。従業員もなし）
- この前提は「報酬が完全に 0 円」であること。将来役員報酬を出すと代表社員 1 名でも強制加入対象になり、社会保険料負担が事業計画に効いてくる
- 役員借入金の返済（自己資金の回収）は報酬ではないので 0 円判定を崩さない

## 添付書類の扱い（重要な気づき）

- 調査票の添付候補 5 種（①法人事業概況説明書 ②役員給与内訳書 ③源泉所得税領収書 ④休眠・休業届 ⑤賃金台帳）は
  **すべて現時点で存在しない** — ①も法人税申告の添付書類なので第1期決算（8/31）未了の今は未作成
- → 添付書類の代わりに **申立書（理由書）** を提出する形式。①〜⑤それぞれ「なぜ出せないか」を 1 項目ずつ潰す構成にすると追加照会が来にくい

## 成果物

- A4 1 ページの申立書 PDF を作成: `~/Desktop/ソルトムーン_申立書.pdf`
- 印刷前の記入欄 4 つ: 提出日 / 年金事務所名 / 所在地・電話 / 代表社員印
- 事業所整理記号があれば右上に手書き付記 or 調査票コピー同封で照合が早い

- 設立以降、不動産取得のみ。未入居で賃料未発生、決算未了（第1期、決算日 2026-08-31）、役員報酬・従業員なし
