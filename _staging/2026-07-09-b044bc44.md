# NoteCreator 立ち上げ — note.com 収益化コンテンツ生産システム（propose→実装→初回リサーチ→分野承認→voicing→企画3案）

セッション期間 2026-07-07〜07-09（cwd: tatoflam/NoteCreator、SessionEnd は 07-21）。
新規プロジェクト **NoteCreator** を構想から実運用初手まで一気に立ち上げた。

## プロジェクトの目的

- バズっている note.com / Threads / X のコンテンツを分析し、課題解決型の有料文章記事を
  リサーチ→分析→企画→構成→執筆→校正→販売→継続更新する一連のシステムを構築する
- **目標: note で月額 30 万円**。継続課金に耐える品質維持・追記更新型の記事を志向
- 派生アイデアとしてココナラ展開も spec 内で検証
- 投稿は手動で構わない（bot 対策を兼ねる）

## 決定事項・設計判断

- OpenSpec change **`note-content-monetization-system`**（proposal / design / specs 8 本 / tasks、validation 通過）
- **private リポ https://github.com/tatoflam/NoteCreator を新規作成し push**（git init から）
- データ収集は規約遵守のため**手動キュレーション主体 + 公式 API のハイブリッド**
  （フルスクレイピング却下、`research crawl` は規約ガードが常に拒否）。Threads API は Phase 3 に延期（data/decisions/threads-api.md）
- human-in-the-loop 維持: 候補の自動レコード化はしない。分野承認は人間の明示操作のみ
