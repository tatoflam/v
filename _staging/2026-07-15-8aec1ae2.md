- **Threads API token 取得の手順知見（05_learn 候補）**: Meta Horizon は VR 用で場所違い、正しくは developers.facebook.com（Meta for Developers）。アプリ作成時に **ユースケース「Threads API へのアクセス」を選ばないと Threads メニューが出ない**。exchange に使う secret は「Threads app secret」（ベーシックの通常アプリシークレットとは別物）。ダッシュボードの「アクセストークンを生成」は最初から long-lived(60日) で出ることがあり、その場合 `th_exchange_token` にかけると "Session key invalid"(code 452) になる → exchange 不要、そのまま `graph.threads.net/v1.0/me` で疎通確認。
- **楽天アフィリ ID 取得知見**: affiliate.rakuten.co.jp、ID は 1 楽天会員につき 1 つ・規約同意で自動発行（審査なし）。アカごとに新楽天会員を作る（isolation）。サイト情報に Threads URL 登録必須（未登録だと成果非承認）。自己購入は規約違反。短縮 URL 自動 refresh 用に webservice.rakuten.co.jp の Web Service App（warmup_w3 までに）。
- persona セッション初版を作成・push: statement「お取り寄せメモ — また買った/一回で終わった/次に狙ってる」、voice の核 = **リピート記録**（interior の「試行錯誤」に対応する信頼軸）、締めは「次はこれ試す」宣言、genre 固有 NG = 健康効能表現（薬機法 + 旧 BAN の健康クラスタ回避）。hooks 6 カテゴリ 20 本、rotation 14 枠。**ユーザー承認待ち**（voice 人ゲート 3 点: voice OK か / bio 人柄行 / 贈答ネタ実体験有無）。
