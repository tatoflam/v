# 家庭用エアコンの省エネ規制 — トップランナー / 区分(い・ろ・は) / 建築物省エネ法

## Summary

東京都内で新築WRC（壁式鉄筋コンクリート造）物件を継続的に建てる建設会社が、ハイセンス製エアコン（低価格帯=区分「は」相当が多い）を 2026 年 5 月以降も「海外メーカーだからトップランナー制度の対象外」というロジックで使い続けられるか、を調査した結果。

**結論: 「海外メーカー免除論」は法的に不成立。** ただし「区分（は）製品の使用可否」と「トップランナー達成義務」は別レイヤー。以下が実務上の急所:

1. **海外メーカーも対象**: 省エネ法トップランナー制度は「製造事業者等」に**輸入事業者**を含む。ハイセンスジャパン株式会社（事業内容「家電の輸入・販売等」）は制度上の射程内。勧告 → 公表 → 命令 → 罰金。
2. **区分「は」は建築物省エネ法側の話**: JIS B 8615-1 の定格冷房 COP/EER で（い）（ろ）（は）を判定。住宅**仕様基準ルート**では (ろ) または (い) のみ可、**(は) は仕様基準不適合**。標準計算ルートに切替えれば (は) でも建物全体 BEI で挽回可能。
3. **「2027 年問題」の本質**: 2027 年 4 月から壁掛形エアコンに新トップランナー基準が適用、4.0kW で最大 +34.7% の効率改善要求。ハイセンス現行 S シリーズは新基準達成率 ~74%、H シリーズ ~87% で、出荷継続には設計変更が不可避 → 価格上昇 or 撤退 or 高効率機抱合せ。
4. **2026/5 〜 2027/3 の取扱い**: 旧基準（2010 年度目標）が引き続き有効。区分「は」製品の調達・設置自体は合法。ただし 2026/4 からの建築物省エネ法改正で中規模非住宅 BEI が 0.75〜0.85 に強化済 → (は) を使うと外皮性能 / 窓 / 給湯で帳尻合わせコストが膨張する。

**実務的着地点**: 標準仕様は「(ろ) 以上」を最低ラインに置き、(は) は性能基準ルートで通る案件のみ個別判断。ハイセンスを継続採用する場合は、(ろ) 相当の型番（M シリーズの 2.2kW/2.8kW 等）に限定するか、2027 年度新基準対応機への置換計画をベンダーと事前協議する。

## 法制度の二層構造

```
機器側規制 (省エネ法 / トップランナー制度)         建築側規制 (建築物省エネ法)
─────────────────────────────────────         ─────────────────────────────
対象: 製造事業者等 + 輸入事業者                  対象: 建築主 / 設計者
評価: 年度出荷の加重調和平均で APF 基準値達成    評価: 一次エネルギー消費量 (BEI)
ペナルティ: 勧告→公表→命令→罰金                 ペナルティ: 確認申請通らず / 完了検査済証出ず
施行: 壁掛形は 2027 年度、他は 2029 年度          施行: 2025/4 全新築義務化、2026/4 中規模強化
指標: APF (JIS C 9612:2013)                       指標: 区分(い)(ろ)(は) ← JIS B 8615-1 定格冷房 COP
```

両者は**別制度**。「ハイセンスがトップランナー達成」≠「型番が(ろ)」、「型番が(は)」≠「トップランナー未達」。

## 区分(い)(ろ)(は) の閾値

定格冷房能力 ÷ 定格冷房消費電力 (W/W、JIS B 8615-1) で判定。

| 定格冷房能力 | (い) | (ろ) | (は) |
|---|---:|---:|---:|
| 2.2kW 以下 | 5.13 以上 | 4.78 以上 5.13 未満 | 4.78 未満 |
| 2.2kW 超〜2.5kW | 4.96 以上 | 4.62 以上 4.96 未満 | 4.62 未満 |
| 2.5kW 超〜2.8kW | 4.80 以上 | 4.47 以上 4.80 未満 | 4.47 未満 |
| 2.8kW 超〜3.2kW | 4.58 以上 | 4.27 以上 4.58 未満 | 4.27 未満 |
| 3.2kW 超〜3.6kW | 4.35 以上 | 4.07 以上 4.35 未満 | 4.07 未満 |
| 3.6kW 超〜4.0kW | 4.13 以上 | 3.87 以上 4.13 未満 | 3.87 未満 |
| 4.0kW 超〜4.5kW | 3.86 以上 | 3.62 以上 3.86 未満 | 3.62 未満 |
| 4.5kW 超〜5.0kW | 3.58 以上 | 3.36 以上 3.58 未満 | 3.36 未満 |
| 5.0kW 超〜5.6kW | 3.25 以上 | 3.06 以上 3.25 未満 | 3.06 未満 |
| 5.6kW 超〜6.3kW | 2.86 以上 | 2.71 以上 2.86 未満 | 2.71 未満 |
| 6.3kW 超 | 2.42 以上 | 2.31 以上 2.42 未満 | 2.31 未満 |

判定式（国交省告示式）: 暖房 `−0.321×能力[kW] + 6.16` 以上、冷房 `−0.504×能力[kW] + 5.88` 以上 = (ろ) ライン。

## 仕様基準 vs 標準計算（住宅）

**仕様基準ルート**:
- ルームエアコンは (ろ) または (い) 必須。**(は) は不適合**（国交省 FAQ 明示）。
- 共同住宅は全住戸が住戸単位で仕様基準適合する必要あり。
- 同一室に複数台設置時は (は) > (ろ) > (い) の**悪い方優先**で評価 — 一部だけ高効率にしても (は) が混ざると引っかかる。
- 必要書類: 納入仕様書 / 品質証明書 / 施工記録書を現場備付、完了検査で型番確認。
- 未設置引渡しの場合: 図面・仕様書に「入居後に設置する旨」記載、許容型番 or 「(ろ) 以上」性能条件を契約書に落とす。

**標準計算ルート**:
- Web プログラムに実機性能を入力、住棟単位で一次エネルギー基準達成を判定。
- (は) でも建物全体で BEI 基準を満たせば適合。
- 1 住戸でも標準計算を使うと建築物全体で省エネ適判が必要。
- 設計図書 / 計算書 / Web プログラム出力等の書類負担が増える。

東京都の 23 区・多摩は省エネ地域区分 6 地域（**寒冷地仕様ではない**）。

## 中規模非住宅 BEI 基準引上げ (2026/4 〜)

| 用途 | 〜 2026/3 | 2026/4 〜 | 2030 年度目標 |
|---|---|---|---|
| 工場 / 事務所 / 学校 | BEI ≤ 1.00 | BEI ≤ 0.75 | BEI ≤ 0.60 |
| ホテル / 百貨店 | BEI ≤ 1.00 | BEI ≤ 0.80 | BEI ≤ 0.70 |
| 病院 / 飲食店 / 集会所 | BEI ≤ 1.00 | BEI ≤ 0.85 | BEI ≤ 0.70 |
| 小規模 (300㎡未満) | 説明義務 | 適合義務化 (2025/4〜) | BEI ≤ 0.80 |

`BEI = 設計一次エネルギー消費量 / 基準一次エネルギー消費量`、小さいほど省エネ。

## 2027 年度トップランナー基準 (壁掛形)

| 冷房能力 | 現行 APF 目標 | 2027 年度新目標 | 改善要求率 |
|---|---:|---:|---:|
| 2.2kW (6畳) | 5.8 | 6.6 | ~14% |
| 2.8kW (10畳) | 5.8 | 6.3 | ~9% |
| 4.0kW (14畳) | 4.9 | 6.6 | **~35%** |

壁掛形以外・マルチタイプは 2028 年度まで旧基準が続き 2029 年度から新基準。

## ハイセンス現行ラインナップの状況 (2026/5 時点)

- **S シリーズ**: 主力 4 機種すべて (は) 相当。2027 年度基準達成率 ~74%。
- **M シリーズ**: 2.2kW / 2.8kW 機は (ろ) 相当。仕様基準ルート可。
- **H シリーズ (住設モデル)**: 2027 年度基準達成率 ~87%。
- **付加価値機能**: 解凍洗浄 (経年劣化抑制で電気代 −9% 訴求)、外気温 50℃対応、R32 冷媒採用。
- **法的位置付け**: ハイセンスジャパン株式会社は CSR レポートで事業内容を「家電の輸入・販売等」と公表 → 省エネ法上の輸入事業者として制度射程内。
- **500 台閾値の細い論点**: 勧告・命令の対象要件は「国内向け年間生産量 or 輸入量 500 台以上」。下回れば直接の勧告対象外だが、それは建築物省エネ法上の (は) → (ろ) には**つながらない**（別制度）。

## 三者の見解比較 (要旨)

| 観点 | Claude | Gemini | ChatGPT |
|---|---|---|---|
| 海外メーカー免除論 | 不成立 (輸入事業者として対象) | 不成立 (法的実態を詳述) | 不成立 + 500 台閾値の細い余地に言及 |
| 区分 vs トップランナー | 別レイヤーとして明示分離 | 別制度として整理 | **最も明確に分離**、APF と COP/EER が別という核心を強調 |
| 2026/5 の使用可否 | 仕様基準ルート不可、標準計算で個別判断可 | 法的には可だが BEI 悪化リスク大 | 仕様基準では (ろ) 以上限定、(は) なら標準計算へ |
| ハイセンス具体型番 | 言及なし | 言及なし | **S=は、M=ろ** を具体的に提示 |
| 結論 | (ろ) 最低ライン、置換計画準備 | (い)/(ろ) 標準化＋早期調達 | ルートと型番の組合せ次第、評価ルート選定が鍵 |

合意点:
- 海外メーカー免除論は不成立。
- 2026/5 時点では区分(は)の調達・設置は法的可能だが、BEI 適合コストや 2027 年以降の供給不安定化リスクが高い。
- 標準仕様は (ろ) 以上、(は) は標準計算ルートで個別判断するのが堅実。

分岐点:
- ChatGPT のみハイセンス具体型番に踏み込み、S=は・M=ろ という調達判断の入口を提示している。

## 御社（建設会社）への提言

新築 WRC 物件の継続建設という前提で:

1. **標準仕様の見直し**: 区分(は)前提の設計を廃止、最低 (ろ) を標準仕様に組み込む。設計図書 / 適判書類のひな型も更新。
2. **ハイセンス継続採用条件**: 同社内でも M シリーズの (ろ) 相当機（2.2kW / 2.8kW）に限定。S シリーズを使うなら標準計算ルートで建物全体 BEI を担保する設計に振る。
3. **2027 年度新基準対応機への置換計画**: ベンダーと事前協議（廃番 / 価格 / リードタイム）。2026 年後半は駆け込み需要と旧基準機の生産終了で品薄リスクが顕在化。
4. **完了検査リスク**: 仕様基準ルートで申請したのに現場で (は) 機が設置されると検査済証が出ない事例あり。施工記録 / 納入仕様書 / 現場写真の管理を強化。
5. **未設置引渡し時の契約書文言**: 「(ろ) 以上」性能条件を売買契約書 / 賃貸仕様書 / 図面に必ず記載。
6. **設計初期からの設備スペース確保**: 高効率化 → 室内機 / 室外機の大型化と配管経路余裕が必要。

## 株式会社めぐる向け切替判断レポート (2026-05-15 集約版)

3 者照合の翌日に user が同テーマを「**標準計算ルート前提**」の切替判断レポート (HTML) に再構成し、wiki inbox に投入。前項までの法制度整理を踏まえた **コスト試算 × 段階移行 × 社内ルール化案** で、社内承認向けに体裁を整えた最終形。要点を以下に吸収する。

### 前提の再固定
- 株式会社めぐるは **標準計算ルート** で省エネ適判を通している前提 → 区分(は)機種は法的に採用可能。
- 主論点は「(は)機種の継続採用是非」ではなく **「その(は)枠をハイセンスに置き換えるべきか」** に絞り込み。
- BEI 計算上は同区分なら国内/海外で差なし → 判断軸は **「機器単価メリット vs 中長期の供給・価格リスク」**。

### コストインパクト試算（短期 vs 中長期）

| 項目 | 金額目安/台 | 備考 |
|---|---|---|
| ハイセンス S vs 国内メーカー普及機 | ▲1〜3万円 | 2.2〜2.8kW帯、販路・数量で変動 |
| 年間調達ボリューム例 | 90万〜270万円削減 | 30戸 × 3台 = 90台想定 |

- 短期メリットの **年間 100〜300万円規模** は、2027 年 4 月以降のハイセンス価格上昇 **+20〜30%** で **1〜2 年で相殺**される試算。
- 為替 **10% 円安** で機器単価メリットがほぼ消失する水準。
- BEI 悪化への代償コスト（外皮・給湯・照明）はメーカー切替で変化しない継続発生コスト。

### 推奨方針 — 3 verdict フレーム

| 方針 | 判定 | 条件 |
|---|---|---|
| ハイセンスへの全面切替 | **非推奨** | 標準仕様依存は 2027 以降の調達リスクが事業継続性に直結 |
| ハイセンスの限定採用 | **条件付で可** | 5 条件 AND 一致（下記） |
| 国内メーカー継続採用 | **推奨** | 普及機に(ろ)相当品も投入されており価格差は数千〜1万円程度 |

**限定採用の 5 条件 AND**:
1. 2027 年 3 月までに竣工・引渡し確定
2. 機器単価差が案件規模で明確にメリット
3. サプライヤーから納期保証を文書取得
4. 代替調達計画（国内メーカー切替シナリオ）準備済
5. メーカー側の 2027 年度以降継続供給意向を文書確認

### 段階移行戦略

| 時期 | 標準仕様 | 補足 |
|---|---|---|
| 2026/5 〜 2027/3 | 国内メーカー(は)継続 + ハイセンス限定採用可 | 標準計算ルートで(は)維持、コスト最適化 |
| 2027/4 〜 2028/3 | 国内メーカー(ろ)への移行開始 | 新基準対応機の価格動向見ながら段階移行、BEI 改善で外皮・給湯仕様の見直し余地 |
| 2028/4 以降 | 国内メーカー(ろ)以上を標準化 | 市場全体で(は)機種供給縮小、(ろ)以上が事実上のスタンダードに |

### 社内ルール化案 (運用チェックリスト)

ハイセンス採用案件で **全項目 Yes** の場合のみ承認、1 項目でも No なら国内メーカー採用:

- [ ] 2027 年 3 月までに竣工・引渡しが確定しているか
- [ ] 機器単価差のメリットが案件規模で明確か
- [ ] サプライヤーから納期保証を文書取得しているか
- [ ] 代替調達計画（国内メーカー切替）を準備済か
- [ ] 2027 年度以降の継続供給意向をメーカーに文書確認したか
- [ ] 標準計算ルートでの BEI 計算結果と実機性能の整合確認済か

### 標準計算ルート運用での残余注意点

(全レポートで強調された実務注意、メーカー選定とは独立して常時成立)

- **設計時**: 住宅版 Web プログラムへの実機性能（JIS B 8615-1 準拠）の正確入力、外皮 / 給湯 / 照明 / 換気が BEI 達成可能水準に。
- **完了検査時**: 納入仕様書・品質証明書・施工記録書の現場備付、設計入力値と納入機種の整合担保（型番変更 → 再計算）。
- **機種変更時**: 新機種性能値で再計算、軽微変更で済まない場合は **計画変更の省エネ適判** が必要 → 案件単位ハイセンス切替時はこの再計算コストを織り込む。
- **大型住戸での同一系統複数台**: 評価上「より悪い方」（は > ろ > い 劣位採用）で評価される。
