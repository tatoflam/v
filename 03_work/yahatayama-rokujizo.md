# 八幡山六地蔵案件

## Summary

既存建物の解体を伴う案件。当社は **建設会社の設計PM（元請）** で、**建物の登記名義人ではない**。
2026-04-24 に滅失登記まわりの役割分担を確定し、当社タスクは **建物滅失証明書（解体業者発行）の回収のみ** となった。

## Details

### 案件性格
- 既存建物の解体 → 新築 or 開発フェーズの案件。
- 建物の登記名義人は **お施主様側**（当社ではない）。

### 当社の立場と担当範囲
- 役割: 建設会社の **設計PM**（元請）。
- 解体フェーズで当社が実施するのは **建物滅失証明書の取りまとめ回収** のみ。
  - 解体業者が書類を発行 → 当社経由でお施主様へ。
- **滅失登記の委任状には関与しない**（委任者は建物の表題部所有者＝当社ではない）。

### 2026-04-24 の混乱と解消
- お施主様より「**御社の社判 + 認印 + 捨印** を押した委任状を用意してほしい」旨のメール到着。
- 確認した結果、**建物の所有者は当社ではない** ため、委任関係が成立しないことが判明。
- お施主様に電話確認 → **土地家屋調査士宛の委任状を勘違いして建設会社宛に手配しようとしていた** と判明。
- お施主様に納得いただき解消。当社タスクは建物滅失証明書のみで確定。

### 学び（再発防止）
- 「御社の社判で書類を」と依頼が来たら **まず当社が表題部所有者かを登記事項証明書で確認**。
- 所有者でないのに押印依頼＝**雛形の宛名取り違え** を第一に疑う。
- 関連基礎知識は [[05_learn/metsushitsu-touki-kihon]] に整理済み。
